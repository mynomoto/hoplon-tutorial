Contributing

If you found something wrong or have suggestions about topics that should be covered please open an issue. Just fill the form following this link.

If you have time you can see if someone already complained about the same problem or want more about the same topic looking at list of issues.

You can also usually find me (mynomoto) at the #hoplon clojurians Slack channel. If you are not part of the clojurians Slack, you can request an invitation.
