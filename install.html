Install

Boot

You need to install Boot. You can find installation instructions on Boot's readme. You can go there right now, install it and come back here. I will wait.

If you have any doubts about how to install boot you can reach out on #boot clojurians Slack channel. If you are not part of the clojurians Slack, you can request an invitation.

Hoplon

You don't need to install Hoplon, it is used as a library inside your project. We will generate a project in the next chapter.
